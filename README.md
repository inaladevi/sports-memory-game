# Sports Memory Game

A high-performance, stadium-themed memory card game built using React, Vite, and custom CSS layouts. Test your memory by clicking on unique athlete cards without selecting the same one twice!

## 🚀 Live Demo
You can play the live version of the game here: **[Sports Memory Game Live](https://sports-memory-game.vercel.app/)**

## 🛠️ Features
* Responsive CSS Grid layout optimized for both mobile and desktop screens.
* Retro stadium marquee styling with animated text shadows and glow effects.
* Automatic card shuffling upon user interaction.
